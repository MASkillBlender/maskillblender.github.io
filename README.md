# MASkillBlender Project Page

Static project page for the paper
**"Learning Decentralized Whole-Body Control for Coordinated Multi-Humanoid Loco-Manipulation"**.

Live site: https://maskillblender.github.io/

## Local preview

Either option works:

```bash
# Option A: Python built-in server
python3 -m http.server 8000
# Then open http://127.0.0.1:8000/

# Option B: VS Code Live Server extension
# Right-click index.html -> "Open with Live Server"
```

## Layout

```
index.html                     Main page
static/css, static/js          Vendored Bulma, Font Awesome, Academicons
static/webfonts, static/fonts  Icon fonts
assets/pictures                Teaser image and primitive skill library figure
assets/videos                  Primitive skill and high-level task videos
```
